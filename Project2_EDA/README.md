# Project 2 – Exploratory Data Analysis (EDA)

## Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on the cleaned orders dataset to understand data patterns, trends, distributions, correlations, and outliers.

## Dataset

* Original Dataset: Dataset for Data Analytics (1).xlsx
* Cleaned Dataset: cleaned_orders.xlsx

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Analysis Performed

* Loaded the cleaned dataset
* Displayed dataset information
* Generated descriptive statistics
* Analyzed Order Status distribution
* Visualized Total Price distribution
* Examined Product-wise orders
* Created Quantity vs Total Price scatter plot
* Analyzed Payment Methods
* Calculated Total Sales by Product
* Examined Referral Sources
* Generated Correlation Heatmap
* Created Monthly Sales Trend
* Identified Outliers using Box Plot

## Key Findings

* Dataset contains 1200 records.
* Average Total Price is approximately 1053.97.
* Some products contribute more to total sales.
* Total Price increases with Quantity and Unit Price.
* A few outliers are present in Total Price.
* Different referral sources and payment methods contribute to customer orders.

## Conclusion

EDA helped understand the dataset by identifying trends, distributions, correlations, and outliers. The analysis provides useful business insights and prepares the data for further analytics or predictive modeling.
